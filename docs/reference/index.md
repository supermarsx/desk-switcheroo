# Reference

mitigation details, known limitations, the code's architecture and patterns, how to build
from source, the deployment and runtime lifecycle, and licensing.

These pages document how the app behaves and how it is put together, verified against the
current source rather than marketing copy. For task-focused instructions see the
[Guides](../guides/index.md); for settings see [Configuration](../configuration/index.md).
